HTML Heading

HTML Heading digunakan untuk mendeskripsikan heading (judul). HTML Heading ditampilkan dengan ukuran huruf yang lebih besar dibanding teks biasa. Dimulai dari h1 untuk ukuran huruf terbesa sampai dengan h6 untuk ukuran huruf terkecil.

HTML Horizontal Rule

Elemen ini digunakan untuk memberikan gari horizontal lurus pada halaman web, didefinisikan dengan tag hr.

HTML Paragraf

Elemen ini digunakan untuk mendeskripsikan sebuah paragraf, didefinisikan dengan pasangan tag p.

HTML Line Break

Elemen ini digunakan untuk membeuat baris baru (line break), didefinisikan dengan tag br. Dalam dokumen HTML spasi lebih maupun baris kosong akan dianggap sebagai spasi tunggal, sehingga untuk memberikan line break digunakan elemen break line.
